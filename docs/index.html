Introduction

Creating better quality art using machine learning techniques is imperative for reaching human-like capabilities, as well as opens up a new spectrum of possibilities. How to get started ?

  1. Fork this repository.
  2. Clone the forked repository.
    git clone https://github.com/your-github-username/DeepPixel
  3. Navigate to the project directory.
    cd DeepPixel
  4. Create a new branch.
    git checkout -b your_branch_name
  5. Make changes in source code.
  6. Commit your changes.
    git add .
    git commit -m ""
  7. Push your local branch to the remote repository.
    git push -u origin
  8. Create a Pull Request!

    9. Congratulations! You have just made your contribution to DeepPixel project.

Existing Projects

1. Colorization of black-and-white images

Image colorization is the process of taking an input grayscale (black and white) image and then producing an output colorized image
AIM :- The aim of this task is to generate a script which produces colorized images, when black and white images are provided as input.

2. Style Transfer with Deep Neural Networks

Neural style transfer is a technique which is used to blend together two images - a content image and a style reference image (such as an artwork by a famous painter) seamlessly into one
AIM :- The aim of this task is to transfer the style of the reference image to the content image

3. Face detection

Face detection using Haar cascades is a machine learning based approach where a cascade function is trained with a set of input data.
AIM :- Take an image as input and detect the faces in it. A rectangle is drawn over the detected faces and they are saved as output

4.Denoising Images

Images often get corrupted because of various factors that can occur during acquisition and transmission processes. Image de-noising is the process of removing noise from an image, while at the same time preserving details and structures.
Aim :- The aim of this task is to generate a script which produces denoised images when a noisy image is provided as input.
